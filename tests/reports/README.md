# Test Reports

이 디렉토리는 테스트 실행 결과 리포트를 포함합니다.

## 파일 목록

- `time_system_test_report.json` - TimeSystem 테스트 결과
- `error_handling_test_report.json` - 에러 처리 테스트 결과
- `framework_stabilization_test_report.json` - 프레임워크 안정화 테스트 결과
- `jsonb_validation_report.json` - JSONB 검증 리포트
- `audit_report.json` - 스키마 감사 리포트

## 리포트 형식

모든 리포트는 JSON 형식으로 저장되며, 테스트 실행 시 자동으로 생성됩니다.
