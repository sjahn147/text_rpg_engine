## 🎯 **세계관 이식 전략**

### **핵심 철학**
> **"설정집의 모든 요소가 게임에서 실제로 작동하는 시스템이 되어야 한다"**

- **데이터 중심**: 모든 설정이 DB에 구조화되어 저장
- **상호작용 가능**: 모든 요소가 플레이어와 상호작용 가능
- **역사적 추적**: 모든 사건이 기록되고 세계에 영향을 미침
- **살아있는 세계**: NPC들이 독립적으로 행동하고 진화

## 📚 **세계관 소스 분석**

### **"어둠의 역사적 형식들 설정집 1 - 제국 안브레티아"**

#### **핵심 구조**
1. **정치적 구조**: 안브레티아 제국, 귀족 계층, 관료제
2. **종교적 구조**: The Light Faith, Old Gods, Ancestor Worship
3. **혈통적 구조**: 누한 왕족, 케코와리, 진정한 피 vs 용의 피
4. **지리적 구조**: 레크로스타, 루렌카, 추방자의 산 등
5. **신화적 구조**: 에이레, 루시아, 상징적 객체들

#### **복잡성 요소**
- **다층적 정체성**: 에이레는 여러 시기에 다른 이름으로 등장
- **상징적 객체**: 번성의 잔, 새벽의 망토 등 기능+신화+정치 3요소
- **역사적 해석**: 같은 사건도 관점에 따라 다르게 해석
- **혈통의 진화**: 시간에 따라 변화하는 혈통의 의미

## 🗺️ **1단계: 지리적 세계 구현**

### **1.1 Region-Location-Cell 매핑**

#### **안브레티아 제국 (Region)**
```
안브레티아 제국
├── 레크로스타 (수도) - Region
│   ├── 황궁 - Location
│   │   ├── 접견실 - Cell
│   │   ├── 회의실 - Cell
│   │   └── 개인 서재 - Cell
│   ├── 대시장 - Location
│   │   ├── 식료품 구역 - Cell
│   │   ├── 직물 구역 - Cell
│   │   └── 무기 구역 - Cell
│   ├── 신전구역 - Location
│   │   ├── 빛의 신전 - Cell
│   │   ├── 고대 신전 - Cell
│   │   └── 조상 사당 - Cell
│   └── 하층민 구역 - Location
│       ├── 골목길 - Cell
│       ├── 선술집 - Cell
│       └── 작업장 - Cell
├── 루렌카 (제2도시) - Region
├── 추방자의 산 - Region
└── 기타 지역들...
```
